                        ~~ To create a new MiniGame there are some steps to be followed ~~


                     1. Create the respective folder for the MiniGame inside Games' folder
                     2. Create the respective MiniGame's class extending to the MiniGame class and be sure to follow the implementations.
                     3. Add the game inside GameType enum which is located at Enums' folder
                     4. Create the item for the game inside MGItems located at Enums' folder
                  -------------------------------------------------------------------------------

                                                   IMPORTANT

                    Every file related to the MiniGame should stay inside the MiniGames' folder
                           New Folders are allowed to be created for better organization
